# Changelog

All notable changes to Commando will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-01

### Added
- Initial release
- Command card system with customizable properties (title, number, icon, color, tag, category, description)
- Main view with card layouts (grid and list)
- Terminal view with tab support using Adw.TabView
- Web view using WebKit
- Card editor dialog with right-click context menu
- Speed dial feature (Ctrl+K, type number, Enter)
- Search, filtering, and sorting functionality
- Full keyboard navigation support
- Theme toggle (light, dark, system) in headerbar
- Settings dialog with terminal customization options
- Configurable logging system with multiple levels
- JSON-based data persistence for commands
- Desktop file, metainfo, and flatpak manifest
- Packaging files for deb, rpm, arch, homebrew, and pip
- Comprehensive documentation and API reference

### Features
- Save and organize Linux commands with customizable cards
- Quick execution via speed dial
- Built-in terminal with multiple tabs
- Web browser for documentation
- Search and filter commands
- Full keyboard navigation
- Theme support
